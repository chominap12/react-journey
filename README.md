## 📚 Core Concepts Covered

### 1. JSX (JavaScript XML)
JSX allows writing HTML-like syntax inside JavaScript. It improves readability and makes UI development more intuitive.

---

### 2. Components
Components are the building blocks of React applications. They help in creating reusable and modular UI elements.

---

### 3. Props
Props are used to pass data between components. They make components dynamic and reusable.

---

### 4. State 
State is used to manage dynamic data inside components. When state changes, the UI automatically updates.

---

### 5. Event Handling
React handles user interactions using events such as `onClick`, `onChange`, etc.

---

### 6. Conditional Rendering
Used to render UI conditionally based on logic or application state.

---

### 7. Lists and Keys
Used to render multiple elements dynamically using `.map()` with unique keys for optimization.

---

### 8. Hooks
Hooks are special functions in React that let you use state and other React features inside functional components.

**Common Hooks**

*useState*

Used to store and update data in a component.

👉 Example: counter, form input, toggle
